# 027 Cloud Sync UX Step Clarification

- 日付: 2026-02-17

## 依頼/背景
- クラウド同期の設定画面が、ボタンの役割と操作順序を理解しづらい。
- エンジニアでも挙動の把握が難しく、一般ユーザーではさらに認知負荷が高い。

## 実施内容
- 設定画面のクラウド同期セクションを手順型に再構成。
  - 「使い方（2ステップ）」を明示
  - `1. 同期コードを準備` と `2. データを移す` の2ブロックに分離
- ボタン文言を目的ベースに変更。
  - `強いコードを生成` → `新しい同期コードを作る`
  - `同期コードを保存` → `このコードを保存`
  - `クラウドへ保存` → `この端末をバックアップ`
  - `クラウドから復元` → `バックアップを読み込む`
- 通知文言を操作内容に合わせて具体化。
  - 保存/バックアップ/復元の対象が「この端末」か「クラウド」かを明示
- 入力中コードと端末保存済みコードの差分状態を表示し、未保存状態を可視化。
