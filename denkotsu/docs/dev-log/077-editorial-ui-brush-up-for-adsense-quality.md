# 077 Editorial UI Brush Up For Adsense Quality

- 日付: 2026-04-06

## 依頼/背景
- サイト全体の UI が「AI が量産した雰囲気」に寄って見え、AdSense 審査での品質印象にも悪影響を与えている懸念があった。
- 教育系・知識系サービスの公開サイトを参考調査し、より運営感のある落ち着いた表現へブラッシュアップする必要があった。

## 実施内容
- 参考調査を行い、教育系・知識系サイトに共通する次の方向性を採用した。
  - 過度なガラス表現や発光表現を避ける
  - ファーストビューで価値提案を明確にする
  - 同質なカードの連打を減らし、見出しと区切りを中心に構成する
  - コンテンツ本文を主役にし、UI 装飾を控えめにする
- 全体トーンを再調整:
  - 背景のハロー表現を抑制
  - `panel` 系コンポーネントを半透明ガラスから実体感のある面へ変更
  - 境界線、影、角丸を整理して落ち着いた印象へ寄せた
- ホームを編集し、ヒーローと数値を一体化した編集的レイアウトへ変更。
- ガイド一覧を、カード並列中心から、本文主導の見出し + 区切りリスト構成へ変更。
- ガイド詳細を、入れ子カード中心から、単一の記事面 + 区切りセクション構成へ変更。
- 要点一覧を、カード連打から、カテゴリごとの読みやすいリスト構成へ変更。
- 要点ページの「おすすめ要点」をカード群から分割リストへ変更。
- フッターと下部ナビゲーションを簡素化し、装飾の主張を下げた。
- デスクトップに加えて狭幅ビューポートでも表示を確認し、情報階層が破綻しないことを確認した。
