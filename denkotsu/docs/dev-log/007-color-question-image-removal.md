# 007: 「色を問う問題」からの画像削除

**日付**: 2026-02-16

## やり取りの内容

**ユーザー**: VVFケーブルの色問題のスクリーンショットを示し、「この問題って画像出したら答えわかっちゃいませんか？」と指摘。

## 問題の本質

006で修正したのは**テキストラベル**によるネタバレだったが、今回は**SVGのfill色（視覚表現そのもの）**が答えになっているケース。テキストを削除しても色は画像の本質的な表現であり、削除すると画像の意味がなくなる。

→ 「色を問う問題」に色付き画像を付けること自体が矛盾。画像を外すのが正解。

## 修正内容

以下4問から `image` フィールドを削除：

| 問題ID | 質問 | 正解 | 画像のネタバレ要素 |
|---|---|---|---|
| q075 | PF管の色は？ | グレー（ベージュ） | PF管がfill `#c8c0b0`（グレー）で描画 |
| q109 | VVF 3Cの心線の色は？ | 黒・白・赤 | 心線がfill黒・白・赤で描画 |
| q115 | 圧着工具の柄の色は？ | 黄 | 柄がfill `#f1c40f`（黄）で描画 |
| q119 | テストボタンの色は？ | 赤 | ボタンがfill `#e74c3c`（赤）で描画 |

## 判断基準

「色を問う問題」は純粋な知識問題であり、画像なしでテキストのみの方が適切。SVG自体は他の問題や要点カードで引き続き使用されるため削除不要。

- `src/data/questions.json` - q075, q109, q115, q119からimage削除
